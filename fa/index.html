هاسکال

هاسکال یک زبان برنامه نویسی همه منظوره و متن باز است که برای ساخت نرم افزار بهینه، قابل نگهداری، قابل اعتماد و کارآمد طراحی شده است.

شروع کنید » هاسکال را نصب کنید

آخرین نسخه : v1.4.0 

// مثال مدیریت دستی حافظه در هاسکال
struct point {
    var x : int?
    var y : int?
}

function main() : int {
    // تعریف و تخصیص حافظه متغیر در حافظه هیپ
    var foo : point^ = new point(point(10,10))
    
    // چاپ محتویات متغیر
    print(^foo) // Output: point(x:10,y:10)

زبانی ساده و آسان

هاسکال دارای یک نحو ساده و آسان و کتابخانه استاندارد کوچک است، شما می توانید هاسکال را در کمتر از یک ساعت یاد بگیرید!
هاسکال بسیار شبیه به سوئیفت است و بر اساس ایده های خوب زبان پاسکال در ابتدا پایه گذاری شده است.


ایمن و قابل اطمینان

هاسکال از یک سیستم Null safety برای جلوگیری از خطاهای مربوط به داده NULL استفاده میکند تا موجب افزایش امنیت و پایداری برنامه ها شود.
هاسکال به صورت پیشفرض شما را ملزم میکند تا مقداری را برای متغییر و ثوابت خود(هنگام تعریف) انتخاب کنید تا مطمئن شود هیچ گونه داده NULLی در برنامه وجود نخواهد داشت.


مدیریت حافظه دستی

هاسکال از مدیریت حافظه به صورت دستی استفاده میکند و به شما آزادی عمل در برابر حافظه را میدهد اما هنگام کامپایل شما را در مدیریت حافظه به صورت دستی راهنمایی میکند(WIP).


در هرجایی از هاسکال استفاده کنید

هاسکال، چندسکویی است، به این معنی که در هر پلتفرمی میتواند اجرا شود، از لینوکس و ویندوز گرفته تا سیستم های نهفته!
هاسکال می تواند کد ++C را فراخوانی کند و فراخوانی کد هاسکال در هر زبانی که بتواند با ++C قابلیت همکاری داشته باشد امکان پذیر است.
