The Hascal Programming Language

Hascal is a general-purpose open source programming language that makes it easy to build simple,optimal, reliable, and efficient software.

Simple and Easy Language

Hascal have a simple and easy syntax and standard library, you can learn hascal in 15 minutes!

Hascal is very similar to Swift, Typescript and based on good ideas of Pascal language.

If you know swift, you already know ≈60% of Hascal.


Fast and Powerful

Hascal compiles to native code with dmd and you can use garbage collectio or manual memory allocation for better performance.


Use Hascal every where

You can use D\C\C++\Objective-C to develop your Hascal program,and you can use Hascal functions and structs can use in c\c++ programs.
