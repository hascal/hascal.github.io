The Hascal Foundation


The Hascal Foundation is an open-source foundation was created in 2021 to fund long term Hascal development

The goals of the Hascal Foundation are:

  • To support the whole Hascal ecosystem by funding core technologies
  • To help the Hascal Open Source Community by organizing events and user groups


Supported Projects

The following software is supported by the Hascal Foundation:

  • The Hascal Compiler
  • The Hascal Standard Library
