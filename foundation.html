The Sacalon Foundation


The Sacalon Foundation is an open-source foundation was founded in 2021 to fund long term Sacalon development

The goals of the Sacalon Foundation are:

  • To support the whole Sacalon ecosystem by funding core technologies
  • To help the Sacalon Open Source Community by organizing events and user groups


Supported Projects

The following software is supported by the Sacalon Foundation:

  • The Sacalon Compiler
  • The Sacalon Standard Library
  • Sacalon's Docker image
